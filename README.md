### 활용 기술/기법
1. C의 조건문(if/switch) 
```C++
case 2: //젤리피일때
      switch (attackStyle) // 사용한 공격
      {
      case 1:
         if (sceneNo == 1)
         {
            cout << "SYSTEM : 젤리삐(이)가 건방지게 침을 밷는다." << endl;
         }
         else if (sceneNo == 2)
         {
            cout << "SYSTEM : 산성이었다. 아프다. " << endl;
         }
         break;
      case 2:
         if (sceneNo == 1)
         {
            cout << "SYSTEM : 젤리삐(이) 줄기 채찍을 휘둘렀다." << endl;
         }
         else if (sceneNo == 2)
         {
            cout << "SYSTEM : 아프지만 묘하게 기분이 좋다. " << endl;
         }
         break;
```
2. 함수 작성 및 전방선언
